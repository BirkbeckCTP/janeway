• About
    • The contents of About will appear as a CMS style page. If it is left blank this page will not appear.
  • Logo
    • Your logo will appear in the top left hand corner of the front end. We recommend it be landscape and around 80px in height. If you do not add a logo the Repository Name will be displayed in its place.
  • Footer
    • This field allows you to customse the footer of your repsitory.
  • Login Text
    • This field allows you to inject text into the top of the Login form.
  • Custom JS Code
    • This field allows you to inject any custom Javascript code into the footer area of your site.
